# TranslateThis 

Backend server for translateThis using Anthropic Claude API.

### Install the dependancies:
```
npm install
```

### ENV Variables:

Add the following env variables: 
```
SECRET_KEY=
MONGODB_URI=
PORT=3333
DOMAIN=http://localhost:5173
ANTHROPIC_API_KEY=
```

### Run the server
```
node app.js
```
